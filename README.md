# GIS & Spatial Analysis

В данном репозитории представлены технические и аналитические отчеты по моим проектам в области геопространственного моделирования:

1. **Ykb_Spatial_GIS.pdf**

Анализ пространственного распределения туристической инфраструктуры и мест притяжения г. Екатеринбурга. Включает комплексную оценку пешеходной доступности, плотности культурных объектов и обеспеченности туристических маршрутов санитарной инфраструктурой.

2. **Moscow_KRT_GIS.pdf**

Технический отчет по геопривязке и оцифровке территории проекта комплексного развития (КРТ) в г. Москве. Описывает процесс трансформации растровых изображений (метод Polynomial 1) и создания векторных картосхем.

3**Gagarinsky_Bike_Parking_GIS.pdf**
   
Анализ микромобильности и доступности велосипедной инфраструктуры в Гагаринском районе г. Москвы.
